### <Badge text="POST" variant="tip" size="large"/> /api/dashboard/appointments/cancel
This endpoint allows you to cancel an existing appointment.<br/>
An appointment can be canceled if it has already been approved or not.
:::note
Upon appointment cancellation, an email will be sent to the creator and all members.<br/>
Additionally, if the appointment has previously been approved and the Google Calendar integration is setup then the event will be deleted in the calendar.
:::


### Security
This endpoint requires authentication. You must include a valid API key in the request header.
**Permission(s) Required:** <Badge text="VIEW_APPOINTMENTS" variant="caution" /><br/>


### Request Body
```json
{
    "id": string
}
```
<details>
    <summary>id</summary>
    UUID of the appointment.
</details>
